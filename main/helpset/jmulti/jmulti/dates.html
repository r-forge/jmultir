Numbers, Dates, Variable Names

Numbers

All real numbers are stored as 8-byte double precision values. Missing values are coded with NaN. Number input with exponential notation (e.g. 1.1234e-2) is possible.

Dates

The following date formats are possible:
  • quarterly: "1960 Q2"
  • monthly: "1960 M10"
  • annual: "1960"
  • semi-annual: "1960 I" or "1960 II"
  • daily (5 day week): "11/25/2004 (5)"
  • daily (7 day week): "12/11/1982 (7)"
  • undated: "1"
  • arbitrary spec: "1960 30/265"
Dates can always be coded in the form MAIN_PERIOD SUB_PERIOD/FREQUENCY, e.g. "1950 3/4".

Variable Names

Variable names can contain letters, digits and '_'. All other characters as not allowed and will result in an error message. A variable name must start with a non digit.
