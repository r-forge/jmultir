Time Series Calculator

The TS Calculator can be used to create new variables by combining existing ones with arithmetic expressions. It provides the simple language TSCalc that operates with one dimensional arrays and scalars. Examples are:

newvar = invest + cons;
invest(-1) = lagn(invest,-1);
invest_std = stand(invest);
z = rndn(100);
z;

The code that was used for the generation of new series is used as a description of new variables. This description can be accessed via the SHOW DESCRIPTION item from the popup menu over the time series in the selector. This way it is possible to keep the information on how a set of series was created.

The operation is as follows:

  • First select the variables to be used in the selector. The names appear in the list AVAILABLE VARIABLES and are put into the variable space of the calculator.
  • Write one or more commands to the command area an execute them with EXECUTE or Ctrl+E. Newly created variables appear in the list of available variables.
  • Finally add the selected variables in AVAILABLE VARIABLES to the workspace with ADD SELECTED.

Variable names must start with a non digit and may contain letters, digits and '_'. They are case insensitive. Define a new variable with newvariable = some expression ;. One can execute several commands at once by separating them with ;. If a variable should be printed to the output window, just write variablename;. All array operations are applied element wise. TSCalc understands exponential notation, e.g. 1.234e-3.

The following functions are implemented, where a, b are always a one dimensional arrays, e.g. a variable name, and c is a scalar:

  • max(a,b) - takes 2 vectors of same dimension and returns vector with max elements
  • min(a,b) - takes 2 vectors of same dimension and returns vector with min elements
  • log(a)
  • lagn(a,c) - lags vector a by c periods and fills missings with NaN values
  • fracdiff(a,c) - fractional differencing operator (Hosking 1981)
  • stdc(a) - returns std. deviation of vector a
  • meanc(a) - returns mean of vector a
  • stand(a) - standardizes vector a by subtracting the mean and dividing by its std. deviation
  • reverse(a) - reverses the order of a given vector, starting from observation T to 1
  • rndn(c) - generates a c x 1 vector of random variables between 0 and 1
  • ones(c) - generates a c x 1 vector of ones
  • trend(c) - generates a c x 1 trend vector starting from 1 to c
  • rows(a)
  • sqrt(a)
  • sin(a)
  • cos(a)
  • tan(a)

By double clicking on a variable in the list, the name appears in the command window and can be combined with commands from the calculator. In case there are syntax errors, a descriptive message is printed to the output window. One should correct the error and simply try again. If the selection in the time series selector is changed, the workspace of the calculator is overwritten. Variables that have not been added to the workspace before are lost.
