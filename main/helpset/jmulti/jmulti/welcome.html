JMulTi - Help System

About JMulTi (www.jmulti.com)

JMulTi is a JAVA interface to various Gauss procedures for econometric analysis. It is not necessary to have Gauss installed to run JMulTi because the Gauss Runtime Engine is used which is shipped with the program. JMulTi was created with the JStatCom software framework, see www.jstatcom.com.

How to use this Help

The navigator bar on the left provides a table of contents and a full text search engine.
  • Use the table of contents if you use JMulTi for the first time and want to get an overview of its features as well as to get familar with carrying out basic tasks.
  • Use the full text search engine to look up keywords in all underlying files. The results of the search are ranked after the number of matches.
