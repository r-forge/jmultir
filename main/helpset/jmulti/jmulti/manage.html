Managing Time Series

General Remark

The selection panel provides access to various functions to manipulate and handle the loaded time series.

Edit Series

The time series editor can be used to view and change data points of one or more selected series.

Set Property

Every series can be either endogenous, exogenous or deterministic. This affects how the series is treated by the underlying econometric methods. The property of one or more selected series can be changed via the RIGHT click menu.

Plot

To plot one ore more series from the time series list, just RIGHT click over the selected series and select PLOT. All series are plotted in the same window. Every series is plotted with all datapoints.

Transform

See Transforming Time Series.

Create

See Creating new Series.

Delete

To delete one or more series, select them and either press DEL or RIGHT click over them and select DELETE.

Rename

Select one series, RIGHT click and press RENAME. See variable names for the naming conventions.

Export

See Exporting Data.
