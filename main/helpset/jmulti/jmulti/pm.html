Project Management

Description

Project management in JMulTi makes it possible to store data, output, and the model settings for each module in a single file. This way it is possible to restore the state of an analysis conveniently. JMulTi project files are in an XML format with the suffix .jsc. Projects that have previously been loaded and the currently loaded project appear in the explorer panel. This allows to switch quickly between them by a double click.

Open and Save

After a project has been loaded, it appears in the explorer panel tagged with the symbol. To open a project there are different possibilities:

  • drag and drop project files to the explorer bar
  • use the button from the FILE menu or the toolbar
  • double click on an existing project in the explorer bar

To save the current state of the analysis to a project file, the SAVE AS button should be used. A configure dialog will appear.

It is also possible to save the analysis to an already existing project with its RIGHT mouse popup and the SAVE CURRENT STATE HERE option.

Configuring Projects

The project configuration dialog appears when the SAVE AS button is pressed or when the EDIT item is selected from a projects RIGHT mouse popup. Adjustable settings:

  • project name: the project appears in the explorer panel with that name, it does not have to be unique
  • file: the file to which the project is stored, it always gets the suffix .jsc even if not specified, if a file is chosen that already belongs to another project, that other project will be removed from the explorer panel and is overwritten with the new project
  • project description: an optional description for the project
  • data description: an optional description for the data that is stored with a project, there might be different data types in the future, currently only Time Series Data is available, the default entry are the data descriptions of all imported series


Managing projects

All recently opened projects appear in the explorer bar. This should make it very easy to switch between different analyses. Elements from this list can be deleted by selecting them and pressing the DEL button or by using the RIGHT mouse popup menu. New projects can be put to the list and are instantly loaded via drag and drop.
